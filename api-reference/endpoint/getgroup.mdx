---
title: "Group"
api: "GET http://api.rectify.international/v1/getgroup/:guildid"
description: "This endpoint gets the roblox group information from the Discord Guild."
authMethod: "none"
---

### Body

This request does not use a body, it uses params in the url, replace [:guildid] guild id (Right click the discord server and click Copy ID, you need Developer mode on).

### Response

<ResponseField name="status" type="number">
  Indicates whether the call was successful. 200 if successful, no response if not.
</ResponseField>

<ResponseField name="data" type="object">

Data object containing the user's information.

<Expandable title="Response Info">

<ResponseField name="version" type="string">
  This is the version the API is running on, usually would be PROD(version as an integer).
</ResponseField>

<ResponseField name="success" type="boolean">
  This responds whether the request was successful or not. If the user has been verified before and is able to have their info displayed, it is a boolean value.
</ResponseField>

<ResponseField name="groupid" type="integer">
  This is the integer of the roblox group's id. Example: 123456789
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'https://api.rectify.international/v1/getgroup/1140396813602525214 \
--header 'Content-Type: application/x-www-form-urlencoded' \
```

</RequestExample>
