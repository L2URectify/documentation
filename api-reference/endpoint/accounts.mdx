---
title: "Get Accounts"
api: "GET http://api.rectify.international/v1/user/accounts/:discordid"
description: "This endpoint gets the account information on a user from their discord id."
authMethod: "none"
---

### Body

This request does not use a body, it uses params in the url, replace 1  with the user's roblox id.

### Response

<ResponseField name="status" type="number">
  Indicates whether the call was successful. 200 if successful, no response if not.
</ResponseField>

<ResponseField name="data" type="object">

Data object containing the user's information.

<Expandable title="Response Info">


<ResponseField name="success" type="boolean">
  This responds whether the request was successful or not. If the user has been verified before and is able to have their info displayed, it is a boolean value.
</ResponseField>

<ResponseField name="accounts" type="array">
  Data array containing the user's connected account(s).

</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request GET 'http://api.rectify.international/v1/user/accounts/12345" \
--header 'Content-Type: application/x-www-form-urlencoded' \
```

</RequestExample>

<ResponseExample>

```json Response
{
    "status": 200,
    "data": {
        "success": true,
        "accounts": {}
    }
}
```

</ResponseExample>
